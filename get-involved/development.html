Resolve existing Issues by submitting suggested code changes using Pull Requests. Read the CONTRIBUTING.md file for information on branching, standards, and practices in our code repositories.

Bugs and Maintenance

Help MyBB find and resolve issues with the software

The Bugs and Issues subforums are where defects found in MyBB can be documented by users with no GitHub account or less technical experience. Later, they are pushed to GitHub by the MyBB Team, where solutions and expected target versions can be tracked.

More information on reporting bugs and issues →

Problem Reports

Create bug reports on the Community Forums for the appropriate version of MyBB within the Development category, if the problem was not reported already on the Forums or as Issues on GitHub. If it is, but receives little attention, you can share details on how it impacts your forum to help diagnose the problem.

Issue Verification

Help developers confirm and prioritize bugs by providing additional information or simply by adding a "üëç" reaction to existing Issues to let them know they are affecting you.
