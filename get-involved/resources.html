MyBB Documentation

Contribute to the MyBB Documentation using GitHub

MyBB uses a GitHub repository to manage the official MyBB Documentation, allowing everyone to easily collaborate and contribute to it. To start, you just need to fork the docs.mybb.com repository and open a Pull Request after making changes and pushing them to your forked repository. Once the Pull Request is reviewed, your changes will be merged and applied on the web pages.

MyBB Website

Contribute to the MyBB Website using GitHub

MyBB uses GitHub Pages to host the MyBB Website for free, allowing the MyBB Community and staff to maintain it. To contribute you just need to fork the mybb.com repository and open a Pull Request after making changes and pushing them to your forked repository. Once the Pull Request is reviewed, your changes will be merged and applied on the website.
