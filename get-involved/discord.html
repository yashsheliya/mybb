MyBB Discord Server

MyBB Staff are seperated in the server by a special username color, and are probably around to talk to as well as other MyBB users.

{% include icon.html icon='discord' %} Join us on Discord

Rules and Tips

  • The Code of Conduct is to be observed at all times,
  • support is not guaranteed here — for complex issues and better reach, create a thread in support sections on MyBB Community Forums,
  • please only speak English within the server,
  • avoid posting long messages, or too many messages in a a row — use a paste bin (like paste.laravel.io) instead to post code and error logs,
  • do not use Discord to report potential security issues — go to mybb.com/security instead and use Private Inquiries,
  • do not report confirmed bugs and problems — go to the Bug Reports forum so that developers can investigate them,
  • if you spot abusive or problematic behavior, you can contact one of the MyBB Team members (orange usernames).
